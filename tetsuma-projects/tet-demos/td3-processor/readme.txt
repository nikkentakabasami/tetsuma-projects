------------------

annotation processors
  Механизм, позволяющий до компиляции находить аннотации и обрабатывать их, генеря при этом новые исходники.
  Не умеет изменять существующие исходники. Lombok делает это хакерским способом.

  Компилятор ищет аннотации в исходниках, затем выбирает annotation processors, которые должны их обработать и выполняет их по очереди.
Если annotation processor создаёт новый исходник - процесс повторяется.
  Когда обработка аннотаций заканчивается - начинается компиляция.
