простой веб-проект, который импортирует js-библиотеки из проекта web-jslibs.

Пример импорта статического контента через overlay
Кроме того, подключает сервлет OkaeriServlet3 из web-jslibs.


Запуск:

1)JettyEmbeddedStarter11
  Пример запуска в jetty embedded режиме:

2)jetty-ee10-maven-plugin

запускается через: 
run as -> maven build
jetty:run
